# Bismut

Tournament management platform. Create and manage tournaments, track scores live, and display results on a beamer.

## Stack

- **Next.js** (App Router, Turbopack)
- **Tailwind v4** + **shadcn/ui**
- **next-intl** (i18n, German/English)
- **TypeScript**

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Features

- Multi-tournament support
- Role-based access: Admin, Referee, Viewer
- Live leaderboard via URL / QR code
- Fullscreen presentation mode
- Mobile-optimized referee view
- Auto-generated schedules with manual override
